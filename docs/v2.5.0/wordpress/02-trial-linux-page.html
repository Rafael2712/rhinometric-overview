🚀 Guía de Instalación Rápida

1

Descarga el Instalador

Descarga el script desde el botón de arriba o usando wget:

wget https://licensing.rhinometric.com:5000/downloads/trial-installer -O rhinometric-install.sh
2

Ejecuta el Instalador

Dale permisos de ejecución y ejecuta como root o con sudo:

chmod +x rhinometric-install.sh
sudo ./rhinometric-install.sh

El script instalará automáticamente Docker, descargará las imágenes y configurará el stack.

3

Configura tu Licencia

Introduce la clave de licencia que recibiste por email:

echo "RHINO-TRIAL-2025-XXXXXXXXXXXX" > /opt/rhinometric/license.key
4

Accede a la Plataforma

Una vez completada la instalación (5-10 minutos), accede a:

  • 🎨 Grafana: http://[TU-SERVIDOR]:3000
  • 🔌 API Connector: http://[TU-SERVIDOR]:8000
  • 🎯 Dashboard Builder: http://[TU-SERVIDOR]:8001
  • 📊 Prometheus: http://[TU-SERVIDOR]:9090

⚙️ Requisitos del Sistema

💻 Hardware Mínimo

  • 8 GB RAM (16 GB recomendado)
  • 4 CPU Cores (8 cores recomendado)
  • 20 GB espacio en disco libre
  • Conexión a internet estable

🐧 Software Requerido

  • Docker 24.0 o superior
  • Docker Compose v2
  • Acceso root o sudo
  • Puertos 3000, 8000, 8001, 9090 disponibles
