💻 System Requirements

For the VM Host (your machine):

  • 💾 RAM: Minimum 16 GB recommended (VM will use ~8 GB)
  • 💽 Disk: 50 GB free (20 GB for VM + operational space)
  • 🔧 CPU: 4 cores minimum (6-8 recommended)
  • 📀 Software: VMware Workstation, VirtualBox, ESXi, Proxmox, or compatible
  • 🌐 Network: Internet connection for license verification

Inside the VM (pre-configured):

  • OS: Ubuntu 22.04 LTS
  • Docker: v24.0+
  • Compose: v2.23+
  • Stack: 12+ orchestrated containers

🚀 How to Use Demo OVA (when available)

  1. Download: Get the rhinometric-demo-2.5.0.ova file (~3 GB)
  2. Import: In VMware/VirtualBox → File → Import Appliance → Select OVA
  3. Configure network: Assign network adapter (Bridge or NAT depending on your environment)
  4. Start VM: Boot the virtual machine
  5. Wait for startup: Stack takes ~3-5 minutes to be ready
  6. Access: Open browser at http://[VM-IP]:3000 (Grafana)
  7. Explore: User: admin | Pass: admin
