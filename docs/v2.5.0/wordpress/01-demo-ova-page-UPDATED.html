💻 Requisitos del Sistema

Para la VM Host (tu máquina):

  • 💾 RAM: Mínimo 16 GB recomendado (la VM usará ~8 GB)
  • 💽 Disco: 50 GB libres (20 GB para VM + espacio operativo)
  • 🔧 CPU: 4 cores mínimo (6-8 recomendado)
  • 📀 Software: VMware Workstation, VirtualBox, ESXi, Proxmox, o compatible
  • 🌐 Red: Conexión a internet para verificación de licencia

Dentro de la VM (pre-configurado):

  • OS: Ubuntu 22.04 LTS
  • Docker: v24.0+
  • Compose: v2.23+
  • Stack: 12+ contenedores orquestados

🚀 Cómo usar la Demo OVA (cuando esté disponible)

  1. Descargar: Obtén el archivo rhinometric-demo-2.5.0.ova (~3 GB)
  2. Importar: En VMware/VirtualBox → Archivo → Importar Appliance → Seleccionar OVA
  3. Configurar red: Asignar adaptador de red (Bridge o NAT según tu entorno)
  4. Iniciar VM: Arrancar la máquina virtual
  5. Esperar arranque: El stack tarda ~3-5 minutos en estar listo
  6. Acceder: Abrir navegador en http://[IP-DE-LA-VM]:3000 (Grafana)
  7. Explorar: Usuario: admin | Pass: admin
