🚀 Instalación en 3 Pasos

1
Descarga e Importa la OVA

• Descarga el archivo rhinometric-demo-2.5.0.ova
• Abre VirtualBox o VMware
• Archivo → Importar servicio virtualizado
• Selecciona la OVA descargada y sigue el asistente

2
Inicia la Máquina Virtual

• Asigna al menos 4 GB de RAM (recomendado 8 GB)
• Configura red en modo Bridge o NAT
• Inicia la VM - el sistema arrancará automáticamente
• Espera 2-3 minutos para que todos los servicios inicien

3
Accede a Rhinometric

• Abre tu navegador en: http://localhost:3000
• Usuario: admin
• Contraseña: rhinometric
• ¡Explora los dashboards y funcionalidades!

⚙️ Requisitos del Sistema
