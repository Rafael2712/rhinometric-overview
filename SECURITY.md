# Security Policy

## Repository Scope

This is a **documentation-only** public repository for the Rhinometric
observability platform. It does **not** contain application source code,
backend logic, or any runtime components.

| Belongs here                | Does NOT belong here               |
|-----------------------------|------------------------------------|
| Architecture docs           | Python / TypeScript source code    |
| Installation guides         | Docker images or build artefacts   |
| `.env.example` (no secrets) | Real `.env` files with credentials |
| Deploy templates            | Private keys, certificates         |
| User manuals                | License keys or secrets            |
